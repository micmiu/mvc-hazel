mvc-hazel

基于 micmiu-web-sh 修改成Maven多模块的构建模式，同时对项目代码进行了重构。 这是一个整合了J2EE的主流技术的开发框架，方便快速搭建项目原型。Hazel是我妻子的英文名字，为了感谢她给我们生了一个萌萌的娃，故把这个框架命名为 mvc-hazel (^o^) 。

Technical Summary :

  • 基于主流的MVC技术： SpringMVC 3.2 + Hibernate 4.1.9 + Tiles 2.2.2
  • 前端是基于Jquery + easyui 整合而成
  • 安全认证框架 shiro 1.2.0
  • Maven 构建工具，开发工具（Eclipse、idea 都可）
  • ...
